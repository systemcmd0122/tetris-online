# TETRIS BATTLE ONLINE (Vue.js SPA)

A multiplayer Tetris game rebuilt with Vue.js 3 and Vite.

## Features
- Real-time Multiplayer (Firebase RTDB)
- Sprint Mode (40 Lines)
- Bot Battle
- User Profiles & Stats (Firestore)
- Global Rankings

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
